# Flutter Widget Teacher with BLoC

## Getting Started
Widget teacher is a learning app for flutter widgets. It is a simple app that shows the widget and its properties. It also shows the code for the widget.

## How to run
- Clone the repo
- Run `flutter pub get`
- Run `flutter run`

## Dependencies
- Flutter
- Dart
- Bloc
